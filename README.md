# GO SRE Challenges

SREになるために実務で必要な技術要素を主にGO言語で実装したポートフォリオです。

## 目的
- **Unix哲学の理解**: パイプライン処理や標準入出力を扱うCLIツールの作成
- **Webサーバの基礎**: ヘルスチェック、グレースフルシャットダウンの実装
- **並行処理**: Goroutine/Channelを用いた非同期処理と排他制御
- **可読性と保守性**: ものレポ構成による管理とドキュメントの整備
